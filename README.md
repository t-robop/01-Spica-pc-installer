# 01-Spica-pc-installer
## Discription
Spicaのやつを自動でインストールするやつです
## Usage

### Windows
- PowerShellを管理者権限で起動
- 以下のコマンドでスクリプトの実行ポリシーを変更
```
 Set-ExecutionPolicy RemoteSigned
```
実行すると以下のようになる．Yで進める．
